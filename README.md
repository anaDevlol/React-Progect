# Little Lemon Table Reservation App

## Overview

This project is a React web application for the Little Lemon restaurant.

Users can:

* Browse the restaurant website
* Reserve a table online
* Select a reservation date and time
* Choose the number of guests
* Select an occasion
* Receive a booking confirmation

## Technologies Used

* React
* React Router
* JavaScript
* HTML5
* CSS3

## Features

* Responsive design
* Form validation
* Dynamic available booking times
* Booking confirmation page
* Unit testing with React Testing Library

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm start
```

4. Open:

```text
http://localhost:3000
```

## Tests

Run tests with:

```bash
npm test
```
